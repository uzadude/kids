# Kids Learning Games

A small collection of browser-based learning games for kids. All games are static HTML and run directly in the browser — no build step required.

**Live site:** https://uzadude.github.io/kids/

## Games

- [Multiplication Game](https://uzadude.github.io/kids/multiply-game.html) — practice multiplication tables.
- [Random Letter](https://uzadude.github.io/kids/random-letter.html) — letter recognition and phonics.
- [Random Two Letters](https://uzadude.github.io/kids/random-two-letters.html) — paired-letter practice.
- [The -S Monster](https://uzadude.github.io/kids/verb-s-game.html) — when to add `-s` to verbs (React + Babel standalone via CDN).
- [Bendix King Radio](https://uzadude.github.io/kids/idan/bendix-king.html) — interactive aviation radio simulator.

## Local preview

Open `index.html` in a browser, or serve the directory:

```
python3 -m http.server 8000
```
